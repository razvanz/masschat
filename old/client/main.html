Вышла программа Chrome Remote Desktop для Android

//20.04.2014 Версия для печати Комментарии

Google выпустила версию программы Chrome Remote Desktop для Android. С ее помощью каждый пользователь Android-устройства может управлять своим персональным компьютером со смартфона или планшета. Чтобы отдать персональный компьютер в удаленное управление смартфону, на ПК тоже требуется установить программу Chrome Remote Desktop, она выпускается для Windows и OS X. Клиент для Chrome под Linux тоже существует, но пользователи жалуются на его некорректную работу. Программа от Google стала одним из многих платных и бесплатных приложений аналогичной функциональности в каталоге Google Play. Такие инструменты выпускает Microsoft и другие компании. Первая версия Chrome Remote Desktop вышла в 2011 году. С тех пор Google постепенно улучшает поддержку разных платформ. Теперь дошло дело и до Android, а до конца года обещают выпустить клиент под iOS (кстати, Microsoft уже сделала это). Приложение передает информацию между Android-устройством и настольным компьютером по защищенному каналу SSL средствами браузера Chrome — одной из программ Google, которую не затронул баг Heartbleed. Для работы необходим аккаунт Google. Владельцы разных аккаунтов могут установить канал связи между собой, если один из них сообщит другому одноразовый код, сгенерированный в программе.
